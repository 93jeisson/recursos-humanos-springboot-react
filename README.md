# Sistema de Recursos Humanos

Aplicación web para la gestión de empleados desarrollada con **Spring Boot** y **React**.  
El sistema permite administrar información de empleados mediante operaciones CRUD.

## Tecnologías utilizadas

- Java
- Spring Boot
- React
- JavaScript
- MySQL
- Maven

## Funcionalidades

- Registrar empleados
- Editar información de empleados
- Eliminar empleados
- Listar empleados
- Gestión de datos de recursos humanos

## Instalación

1. Clonar el repositorio
2. Ejecutar el backend con Spring Boot
3. Ejecutar el frontend con React
4. Acceder a la aplicación desde el navegador
